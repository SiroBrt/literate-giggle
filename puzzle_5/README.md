Stern-Brocot
Da a cada fracción una identificación única en binario a partir de un árbol:

1|       1       |0
 |   1       2   |
 |   2       1   |
 | 1   2   3   3 |
 | 3   3   2   1 |
 |       .       |
 |       .       |
 |       .       |
